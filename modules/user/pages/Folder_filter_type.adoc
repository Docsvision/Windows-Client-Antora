= Тип фильтра

Свойство папки *Тип фильтра* используется при настройке папок типа _Виртуальная_.

Если в систему не были добавлены сторонние расширения, определяющие типы фильтрации, то по умолчанию все пользовательские фильтры будут иметь тип _Стандартный фильтр_.

Изменение, выполненное обычным пользователем, будет применено сразу же после сохранения карточки. Если изменение выполнялось администратором, у обычных пользователей изменения будут применены после перезапуска _{wincl}а_ или обновления дерева папок.
