= Личная папка пользователя

_Личная папка_ назначается пользователю администратором при регистрации в _Справочнике сотрудников_. Данная папка предназначена для работы пользователя с документами и хранения ярлыков карточек.

При открытии программы РМА пользователь сразу попадает в свою личную папку, отмеченную значком image:buttons/Folder_Personal_User.png[image].

Папки, назначаемые сотрудникам в качестве личных (личные папки пользователей) получают тип *Персональная*, указываемый в свойствах папки, где отображается также учетная запись владельца.
