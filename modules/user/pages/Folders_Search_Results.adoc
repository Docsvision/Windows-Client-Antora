= Папка "Результаты поиска"

Виртуальная системная папка *Результаты поиска* автоматически создаётся при первом запуске _РМА_ и содержит результаты последнего выполнения расширенного поиска, в который входят только те карточки, к которым пользователю разрешён доступ.

Папка *Результаты поиска* не может быть удалена.
