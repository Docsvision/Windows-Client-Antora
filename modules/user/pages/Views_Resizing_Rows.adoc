= Изменение размеров строк

[NOTE]
====
Изменение размера строки доступно только в тех представлениях, в настройках которых установлен флаг *Позволять изменять размеры строки*.
====

Чтобы изменить размеры строк таблицы _РМА_ выполните следующие действия:

. Щелкните левой кнопкой мыши по левому краю строки.
. Увеличьте/уменьшите высоту строки, удерживая левую кнопку мыши.
