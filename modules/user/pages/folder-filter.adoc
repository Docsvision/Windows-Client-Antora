= Фильтр

Свойство папки _Фильтр_ используется при настройке виртуальных папок. Тип фильтра определяет поисковый запрос, результатом которого будет содержимое папки. Создание запросов выполняется администратором в РМА.

.Чтобы выбрать фильтр, выполните следующие действия:
. xref:folder-properties.adoc[Откройте] окно свойств папки.
. Нажмите кнопку image:admin:buttons/three-dots.png[Три точки] справа от поля _Фильтр_.
. Выберите из стандартного диалога один из ранее созданных (сохранённых) поисковых запросов.
