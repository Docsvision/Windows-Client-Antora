= Связанная папка

Свойство _Связанная папка_ используется при настройке папок типа _Делегат_. Свойство предназначено для хранения пути к папке, на которую ссылается папка-делегат.

Чтобы изменить делегирующую папку, необходимо в данном свойстве указать путь к другой папке.
