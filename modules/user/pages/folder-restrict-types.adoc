= Ограничение на использование типов карточек для папки

По умолчанию в папке могут быть созданы карточки любых типов и видов из числа доступных в _Справочнике видов карточек_. Однако, при необходимости набор типов и видов для создаваемых карточек в папке может быть ограничен.
