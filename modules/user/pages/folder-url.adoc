= Ссылка

Папку можно настроить таким образом, чтобы при открытии папки вместо представления открывалась определённая веб-страница. По умолчанию веб-страница не задана.

Для использования данного режима необходимо дополнительно установить переключатель _Показывать по умолчанию_ переключатель в положение *URL*.

. xref:folder-properties.adoc[Откройте] окно свойств папки.
. Найдите поле _Ссылка_.
. Введите в поле адрес ссылки.
