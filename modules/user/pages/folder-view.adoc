= Представление папки

Настройка представлений выполняется администратором средствами РМА.

По умолчанию набор используемых для папки представлений может быть xref:folder-restrict-views.adoc[ограничен]. Также в свойствах папки можно настроить представление, которое будет отображаться у пользователя при открытии папки. По умолчанию используется вид _Дайджест_.

.Чтобы изменить представление, которое будет отображаться по умолчанию при открытии папки, выполните следующие действия:
. xref:folder-properties.adoc[Откройте] окно свойств папки.
. Найдите поле _Представление_.
. Нажмите на кнопку image:buttons/triangle-black-down-field.png[Треугольник вниз] справа от поля, чтобы открыть список доступных представлений.
. Выберите нужное представление из числа доступных.
+
Настройка представлений xref:engineer::view-settings-guide.adoc[выполняется администратором].
+
Для папки-делегата в списке будут отображаться представления, которые были разрешены для делегирующей папки.
. Нажмите на кнопку *ОК*.
+
