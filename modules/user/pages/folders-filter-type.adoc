= Тип фильтра

Свойство папки _Тип фильтра_ используется при настройке папок типа _Виртуальная_.

Если в систему не были добавлены сторонние расширения, определяющие типы фильтрации, то по умолчанию все пользовательские фильтры будут иметь тип _Стандартный фильтр_.
