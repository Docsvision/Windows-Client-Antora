= Автоматическое применение фильтра виртуальной папки к карточке

По умолчанию, если у пользователя открыта виртуальная папка и карточка в ней была изменена, фильтр папки повторно не применяется к карточке. Из-за этого до обновления содержимого виртуальной папки в ней может отображаться карточка, данные которой после изменения перестали удовлетворять фильтру этой папки.

.Чтобы включить автоматическое скрытие карточек, данные которых не удовлетворяют фильтру виртуальной папки:
. xref:folders-properties.adoc[Откройте] окно свойств папки, затем нажмите на кнопку *Другие*.
. Установите флаг `*Скрывать карточку, не удовлетворяющую фильтру, сразу после её изменения*`.
+
По умолчанию флаг не установлен.
+
[IMPORTANT]
====
Флаг отображается в настройках только виртуальных папок и только в {wincl}е. В РМА флаг не отображается, функция автоматического применения фильтра не работает.
====
+
. Нажмите на кнопку *ОК*.
