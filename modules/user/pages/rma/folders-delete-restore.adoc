= Удаление и восстановление папки

Удаление папок производится в два этапа: <<preliminary,первичное>> удаление и <<final,окончательное>> удаление.

[#preliminary]
== Первичное удаление папки

include::page$folders-delete-restore.adoc[tags=delete]

Первично удалённые папки можно отобразить для просмотра в xref:rma/interface.adoc#tree[дереве папок] или скрыть при помощи кнопки *Показывать удалённые элементы* на вкладке _Вид_ ленты инструментов РМА, при этом строка с записью об удалённой папке перечеркивается. В папке _Корзина_ удалённые папки не отображаются.

.Чтобы первично удалить папку, выполните следующие действия:
. Выделите её правой кнопкой мыши.
. Удалите папку:
+
* Из контекстного меню команду кнопкой _Удалить_.
* Нажмите на панели инструментов кнопку *Удалить* image:buttons/x-red-on-white.png[Красный крестик].
* Нажмите на клавиатуре клавишу kbd:[Delete].

[#final]
== Окончательное удаление папки

После окончательного удаления восстановление папки невозможно.

Чтобы окончательно её удалить, повторно удалите первично удалённую папку.

[#restore]
== Восстановление удалённой папки

.Чтобы восстановить первично удалённую папку, выполните следующие действия:
. Выделите в xref:rma/interface.adoc#tree[дереве папок] нужную папку.
. Выберите из контекстного меню команду *Восстановить*.
+
[NOTE]
====
При восстановлении папки, удалённые вместе с ней карточки, не восстанавливаются, их придется восстанавливать отдельно.
====
